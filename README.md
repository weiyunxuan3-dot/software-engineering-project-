# HabitFlow 个人习惯追踪器

软件工程课程项目 | 第3组

## 项目简介

HabitFlow是一款帮助大学生养成良好习惯的轻量级
Web应用，提供习惯管理、每日打卡和数据可视化三项
核心功能，帮助用户记录坚持过程、获得正向反馈。

## 核心功能

- 习惯管理：创建、编辑、删除日常习惯
- 每日打卡：一键记录当日习惯完成情况
- 数据可视化：查看连续打卡天数与完成率图表

## 技术栈

- 后端：Python
- 前端框架：Streamlit
- 数据库：SQLite
